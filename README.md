# HotelReservationSystem

A hotel reservation management system developed with ASP.NET MVC to practice building a multi-page web application with database integration.

## 🚀 Features

- Hotel listing
- Room management
- Reservation management
- Customer information
- CRUD operations
- Database integration

## 🛠️ Technologies

- ASP.NET MVC
- C#
- SQL Server
- Entity Framework
- HTML
- CSS
- Bootstrap

## 🎯 Purpose

This project was built to practice developing a hotel reservation management system using the ASP.NET MVC framework while improving database design and CRUD application development skills.

## 📌 Future Improvements

- User authentication
- Online payment simulation
- Email notifications
- Admin dashboard
- Responsive UI improvements
